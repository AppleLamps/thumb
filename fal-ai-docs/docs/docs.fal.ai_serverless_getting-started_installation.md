### [â€‹](https://docs.fal.ai/serverless/getting-started/installation\#option-2:-api-key)  Option 2: API Key

Create an API key at [API Keys](https://fal.ai/dashboard/keys) with the `ADMIN` scope. If creating a key for a team, make sure to select your team on the top left corner.Set it as an environment variable:


Or add it to your shell profile (`~/.bashrc`, `~/.zshrc`, etc.) to persist across sessions.
